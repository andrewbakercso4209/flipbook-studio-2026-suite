## Overview

FlipBook Studio Suite is aimed at animators and visual artists who prefer a dedicated environment for drawing motion one frame at a time. It keeps the workflow straightforward on Windows, macOS, and Linux, making it easy for individuals and teams to stay consistent across the desktop platform they already use.

The application is meant to function without an internet connection, which suits local production setups and disconnected work sessions. Layer management, audio-guided timing, and scriptable automation combine to provide a practical toolkit for shaping and polishing animated scenes.

---

## Features

- Frame-by-frame animation tools for traditional sequence-based workflows
- Multi-layer compositing for combining elements during production
- Onion skinning support for comparing adjacent frames while drawing
- Audio waveform scrubbing for timing animation to sound
- Export options for GIF, MP4, and PNG image sequences
- Script automation with Lua and Python for repeatable tasks
- Offline operation for local use without constant connectivity
- Multi-platform support for Windows, macOS, and Linux

## Usage

A typical workflow looks like this:

1. Start a new animation project.
2. Add frames and draw your key poses frame by frame.
3. Use onion skinning to align motion across neighboring frames.
4. Arrange elements across layers for more complex scenes.
5. Scrub the audio waveform to match timing with dialogue or music.
6. Export the finished result as GIF, MP4, or a PNG sequence.

For automation, run your Lua or Python scripts from the supported scripting interface to repeat common steps or streamline batch work.
